解决多设备的烦恼|极简生活指南

2016-05-19 体验 ， iOS ， Mac

如今我们的生活，已经离不开身边众多的电子设备。我们也花了很多时间在这些设备和其中的软件体验上。生活本该是简单，不该被这些设备所束缚。今天我想来谈谈我关于多设备同步的（伪）极简生活指南。

极简的生活在我的世界中就是 尽量的减少不必要的操作 。忘记在设备使用上的隔阂，使体验更加和谐。

连接，是为了忘记连接

很多人喜欢苹果的设备，一方面是因为他较（有）好（逼）的（格）品质，另一方面是因为苹果在能带来其他品牌无法带来的一体化的体验。有一种说法是说，入了 Mac 的人下一部手机十有八九会是 iPhone。毫无疑问，Mac + iPhone 可以带来十分优秀的无缝连接。

可惜我是一个 Android + Mac 的组合，可我还是十分的追求手机和电脑之间的无缝连接。使用很多工具来提高连接的效率。

有线文件传输我使用 SmartFinder ，一款锤子出品的 Android 与 Mac 文件传输工具。相比 Android 自带的文件管理，SmartFinder 在界面和操作上都爽快了很多。无线的连接我一般使用 AirDroid，不用讲，这是一款大家都熟知并且功能全面和强大的多平台连接应用。一般我使用网页版便足够了。

一些辅佐的连接工具也会提高你的工作效率。Pasteasy 是一款我一直在用并且认为十分优秀的跨平台剪贴板工具，它可以在连接好的在同一 WiFi 下的不同设备之间同步剪贴板的内容，不仅是文字，也包括你复制的文件，所以在一定程度上它可以取代其他连接工具，帮你快速有效的传输文件。

如果你是标准的 iOS+Mac 的用户，那么在连接上会有更多优秀的功能出现。例如 Handoff 可以快速的在另一个 iOS 或者 OS X 上继续你之前设备的操作，比如写邮件、浏览网页。Airdrop 可以传输文件。一些第三方 App 的多平台也会只支持这两平台，比如我在用 Near Lock

全平台，体验无缝生活

很喜欢的一部动漫叫《钢之炼金术师》里有一句话：「一即是全，全即是一」。在多设备使用上，我也崇尚这样的哲学。如果有的选择，我会优先使用有云同步功能、支持多平台的 App。

笔记应用有很多种选择，像是  为知 ， 印象笔记  等等，都支持云同步功能。Todo 类应用你可以选择系统自带的备忘，但是就无法在电脑上进行编辑，也无法在电脑工作时得到及时提醒，所以尽量的选择像  奇妙清单  等全平台覆盖的 App，这样可以提高你的效率。

对文件的同步上，照片的同步我使用 Google Photo，一些 Markdown 文件我会用 iCloud 同步，当然，我更多是使用 Dropbox。如果没有梯子的话，国内网盘目前就剩百度云的网盘算是比较可靠了吧。

这里还有一个使用技巧。一般的日历都会支持互联网账户登录，你可以使用自己的邮箱绑定在两个设备上，这样你的日历操作就会同步到另一个设备上去。不需要额外的下载 App，就能达到多平台同步的效果。

工作流，不做重复的工作

关于 Workflow 的效率神器有很多。在 Mac 上有 Alfred2 ，iOS 上有  Workflow ，Android 上有  IF 。这些软件的功能就是预先设定好经常使用的工作流,用快捷的方式快速的使用操作，减少了不必要的操作。

我的 IF 最早的 Workflow 是同步我 Rss 的星标到 Pocket，无论我在哪个平台使用哪个 Rss 客户端，都可以同步我的星标到同一个地方。另一个是同步我的 Instagram 到我的微博，同步我的微博到我的 Twitter。

Alfred2 最常使用的当然是搜索操作。我先是设置将大多数的搜索方法集中在一起。另一方面，我也会开发一些自己常用的 workflow 比如： chanjh/findFreeEbooks - Github

iOS 上的 Workflow 这个使用起来其实十分之爽。如果能灵活的运用它，会让你在 iOS 上的使用更加的有效率。也许入门可以看看我整理的一些：ios的workflow到底怎么用？。目前我使用比较多的是快速删除最新的截图(在 Android 上留下的老习惯了……)

结语

当你认真思考数字生活和人自身的关系的时候，你就发现其实可以通过更多的工具使得自己拥有更加出色的体验。尽管 iPhone 很优秀，Mac 也很棒，但是如果不懂得使用它们，那就和其他任何的设备没有差别。如果你也有一些极简的生活体验，欢迎你在评论区里面和大家分享。
