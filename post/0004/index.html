Android 上这些 Apps 你不能错过（一）

2015-11-24

推荐一些平时使用的比较频繁的 Android apps，尽量推荐的是只有 Android 平台才有的 Apps。以后会不定期的推荐在不同平台使用的、有趣的 Apps。

本期推荐：Press、Fuubo、IF、锤子便签

##Press

如果你有阅读 Rss 的习惯，那么 Press 这款软件你就不能错过。Press 上优秀的手势操作，在滑动之间完成了返回、收藏、标记为已读等等，操作起来十分流畅，加上支持feedly、自动离线、离线图片的功能，Press绝对是一款优秀的阅读器。 唯一的遗憾可能是 Press 没有搜索功能。

Fuubo

Fuubo 是 Android 上老牌的第三方微博客户端了，记得当年因为支持 2.X 让Fuubo成为我的 Moto 里程碑唯一能用的第三方客户端。 Fuboo 拥有完美的沉浸式状态栏、夜间模式、支持原图上传（大赞）、拥有多种主题……既美观又兼顾了实用性。虽然部分功能因为官方对 api 的限制无法实现，但是在美观等层面 Fuubo 绝对秒杀丑的一逼的官方端。 ##IF（原名ifttt）

IF 原名 ifttt，是「If this then that」的缩写。顾名思义，你可以自定义多种多样的动作，完成自己希望的响应。你的动作可以是：如果更新 ins 的图片，那么同步到 Dropbox、如果天气下雨，那么收到短信、如果靠近你家，那么打开家里的智能家居…… IF 提供的组合方式十分之多，还有热门的设定方式供你选择。我自己最常用的设定是同步我 feedly 收藏夹的内容到 Pocket，这样我在 Press 星标的东西就能在 Pocket 上阅读了。

锤子便签

锤子便签一开始就以简单好用广受好评，而今锤子便签更新到 2.0 ，支持 Markdown 以及网页分享，加上之前的图片分享、云同步等，功能上可谓实用。 作为锤子系的产品，锤子便签也少不了其拟物的风格，从按钮到便签页的设计都绝不马虎。
