我做的一些 Workflow
· 2018-03-09 · # iOS # 自动化 # workflow # app

Workflow 是 iOS 平台上一款明星应用。Workflow 使用了「模块化编程」的思想，我们在 Workflow 里可以快捷、模块化地将一个工作流固定下来。比如你先复制了内容，然后替换了里面的文字，再把文字发送出去或者保存到日历。我在「加上这些干货，让 Workflow 如虎添翼」里介绍了一些 Workflow 周边的工具。

这篇文章将介绍一些我个人做并且常用的 Workflow，可供大家参考使用。
